# Mumbai Village Landing Page

Landing page sederhana untuk Mumbai Village digital.

## Cara Menjalankan
1. Install dependensi: `npm install`
2. Jalankan server: `npm start`
3. Buka browser ke `http://localhost:3000`

## Fitur
- Halaman responsif dengan Bootstrap.
- Form kontak yang terhubung ke back-end (kirim email).
- Struktur folder sederhana.
